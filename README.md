# Predicting Song Popularity Using Machine Learning

## Overview
This project investigates whether audio features from Spotify tracks can be used to predict song popularity. Machine learning models are applied to analyze relationships between musical characteristics and listener engagement.

The project is developed as part of a machine learning course and explores how audio features may help estimate popularity scores.

## Dataset
The dataset contains Spotify track information including audio features and popularity scores.

Example variables include:
- danceability
- energy
- tempo
- loudness
- valence
- popularity

These features are used as predictors for machine learning models.
